# nEDI Platform

An Electronic Data Interchange (EDI) platform for produce vendors, built with Next.js.

## Features

- **Vendor Portal**: Web interface for vendors to manage EDI transactions
- **Admin Dashboard**: Internal tools for managing the platform
- **EDI Processing**: API endpoints for handling EDI files
- **NetSuite Integration**: Planned integration with NetSuite for ERP

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` - Next.js App Router pages and API routes
- `src/components/` - Reusable React components
- `src/lib/` - Utility functions and configurations
